WPVero for WordPress is the easiest way to integrate getvero.com into your WordPress site.

== Description ==

WP Vero is the easier way to integrate Vero in Wordpress.

Using this plugin let you automatically track you users/commenters and their actions without having to change any code on your site.

== Installation ==

1. Go to the **Plugins > Add New** page in your WordPress admin.
1. Search for "WP Vero" and install **WP Vero**.
1. Click **Activate Plugin**.
1. Go to **Settings > Vero Settings** and enter your Vero API key. (You will need a valid Vero account.)

That's it! You can now turn on any analytics service with the click of a button in the Segment.io interface.
